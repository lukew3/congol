# Congol

Conway's game of life for 2 players

[congol.net](http://congol.net)

[Design files](https://www.figma.com/file/2FNvlsHa7aIuhYawCjx0iH/Congol)


## Install Requirements
```
npm ci
```

## Build
```
npm run build
```

## Deploy
Deploys to the gh-pages branch of the repo
```
npm run deploy
```
